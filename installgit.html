Install Git

We installed Git and became familiar with using the terminal. On my terminal I use git bash. Git allows you to have version control over your projects. It allows you to store files and data in your repository. Github is where I store my repositories and together with my command line I save updates to my projects as I work on it.

These were some of my thoughts when we did this project, it is cool how much I have learned in such few weeks. "I have used command line and git before. I have used it for about a year and have ran different commands besides git. Ive used npm to install different things like babel. But I am not completely comfortable using the command line. I still run into problems here and there. I am always learning and I like using terminal as well as the terminal provided in vs code. There are good resources online to help you with running commands but the problem I often have is that I forget which goes first for example when adding, commiting and pushing to github. I have seen the github gui and it is a lot easier to use and pick up the commands but I feel that its not as used by professional developers and perhaps even frowned upon. Using the terminal I think give you more power which is cool and a very good tool."
