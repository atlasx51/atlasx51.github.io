Editors

For this assignment we learned about code editors. Specifically Visual Studio Code. It is a code editor created by Microsoft and is very popular amongst developers.

It is an editor that I have been using for awhile and it also free! It has many extensions, color themes and many other features that enhance the experience of using a code editor.

There are other code editors like atom and sublime text but I don't really like them as much and sometimes feel slower.
