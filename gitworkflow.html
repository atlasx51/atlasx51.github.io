Git Workflow

We became familiar with the Git workflow, in this weeks assignment we updated, added content, merged and pushed files to github. It is once again amazing looking back at the comments how much more familiar I am with all this. Of course there is still a lot to learn but that initial fear isn't as bad.

The following are some of the comments from that weeks assignment. "Overall this assignment was pretty straightforward and very good practice. I feel like I am getting more comfortable with the terminal and git. I did come across some issues with merge conflicts. They were a little frustrating but I quickly thought to my self to take it as a learning experience. I was able to find an answer on stackoverflow and it made sense once I solved the merge conflicts. It may seem at times a simple tasks that we are assignment but they are very useful for us to become comfortable using the commands. I am beginning to like the terminal in VS code, its convenient. What I dont like is my code lighting up with red and yellow tags telling me to add and commit new code. I am not sure yet on how to take that off or if I can but I assume it has to do with VS codes built in version control. "
