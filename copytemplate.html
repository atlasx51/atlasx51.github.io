Once we had chose a template to use for our Project 0 we had to copy our template into our github pages repo. We also went through the files and looked for 3 files that we wanted to research on.

What I researched was sass, gulp and animate.css. This is what I posted that week, "After looking over the code from my template the 3 things that I researched were Sass, Gulp and animate.css Sass is an extension of CSS, it lets you create variables , mixins and nesting css. In their description it says that it helps you keep your css organized especially if its a large style sheet. For example it allows you create a variable and assign it a value, for example $coolblue and whatever color you want $coolblue and you pass it around throughout the style sheet. more info on sass (Links to an external site.)Links to an external site. Gulp is what is called a task runner and toolkit that can be used with anything. It is usually installed with node.js. And its mainly used for tasks that is time consuming or repetitive. For example minification of scripts which is eliminating the white space and shrinking the size of a file. It also can do things like linting which runs the code in search of errors. It seems like it is very popular and when you are more advanced in your coding it is probably very useful it seems more info on gulp (Links to an external site.)Links to an external site. Animate.css is looks like it is a plug in for css animation and effects. If you go to their homepage it gives you examples of different options you can choose, like slide in, bounce, and shake. All these are effects added to your font or can be done to an image. It is pretty cool and it seems like it is easy to use. "
