Cloud

Cloud

The lecture on Cloud computing explained what exactly is the cloud and its benefits.

    Why Cloud?

  • You had to buy a stack of servers before if you wanted to host a website.
  • When you had high traffic you needed more servers.
  • These servers needed a lot of maintainence.
  • They were also high cost.
  • Troubleshooting problems are tedious and may conflict with your business goals.
  • Amount of data is huge and that is why the cloud became important.
  • The reason for the cloud is because we do everything online, bookings, music, files, ebooks, podcasts, etc.
  • Cloud computing is a place to store data on remote servers.

    • Service Models

  • SaaS is software as a service, like Gmail.
  • PaaS is platform as a service, like the google app engine. You can create your own applications on their platform.
  • IaaS is infrastructure as a service, provides virtualized computing resources over the internet, abstract the user from the physical machine.

    • Deployment Models

  • Public cloud is available to everyone.
  • Easy and inexpensive setup, hardware, application and bandwith costs are covered by the provider.
  • No wasted resources because you pay for what you use.
  • Private clouds offer hosted services to a limited number of people behind firewall, so it minimizes the security concerns.
  • Private cloud gives companies control over their data.
  • Hybrid cloud is a combination of private and public cloud.
  • It helps to access the best of both worlds.

    • Cloud Providers

  • Google cloud platform
  • Digital Ocean
  • IBM Cloud
  • Amazon Web Services
  • Microsoft Azure

  • Amazon Web Services is the largest cloud provider, it has flexible pricing
  • Amazon started before other companies
  • Google is one of the cheapest providers in the market
  • Netflix uses Amazon Web Services(AWS)
  • AWS allos for a free tier
